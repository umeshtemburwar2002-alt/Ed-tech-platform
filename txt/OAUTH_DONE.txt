│                                                                                 │
│  STEP 1: Google OAuth (5 min)                                                 │
│  ├─ Go to: https://console.cloud.google.com                                   │
│  ├─ Create OAuth 2.0 credentials                                              │
│  ├─ Add redirect: http://localhost:3001/auth/callback                         │
│  ├─ Copy Client ID & Secret                                                   │
│  └─ Add to Supabase → Authentication → Providers → Google                     │
│                                                                                 │
│  STEP 2: GitHub OAuth (5 min)                                                 │
│  ├─ Go to: https://github.com/settings/developers                             │
│  ├─ Create new OAuth app                                                      │
│  ├─ Set callback: http://localhost:3001/auth/callback                         │
│  ├─ Copy Client ID & Secret                                                   │
│  └─ Add to Supabase → Authentication → Providers → GitHub                     │
│                                                                                 │
│  STEP 3: Test (5 min)                                                         │
│  ├─ Start frontend: npm start                                                 │
│  ├─ Go to: http://localhost:3001/login                                        │
